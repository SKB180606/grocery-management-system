Grocery Management System

Description:
A software application that helps manage grocery store inventory,
billing, and product records.

Features:
- Add, update, and delete grocery items
- Track item quantities and prices
- Generate total bill for customers

Technologies:
- C++
- VS Code

What I Learned:
- Writing modular code
- Using conditionals, loops, and data structures and Object Oriented Programming
- Basic software design and debugging
